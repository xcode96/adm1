CISSP Study Platform

  • Category: Web Development
  • Technology learned:Next.js 14, TypeScript, Tailwind CSS, LocalStorage
  • Project URL: https://cissp.xocode.info/

Details:

A free, privacy-focused CISSP exam preparation platform with 900+ practice questions across all 8 domains. Designed as a static web application, it runs entirely in the browser with all user progress stored locally, requiring no registration and ensuring complete privacy

    Built with Next.js 14, TypeScript, and Tailwind CSS
    No signup / no tracking → privacy-focused design
    900+ practice questions covering all 8 CISSP domains
    Responsive design for use on desktop and mobile devices
